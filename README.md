# AI Time Management Coach

A production-ready Next.js web application featuring an AI-powered time management coach chatbot powered by xAI Grok API.

## Setup

1. Install dependencies:
```bash
npm install
```

2. Create `.env.local` file and add your xAI Grok API key:
```
XAI_API_KEY=your_xai_grok_api_key_here
```

3. Run the development server:
```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser.

## Features

- 🤖 **AI Chatbot** — Powered by xAI Grok, focused on time management & productivity
- 📊 **Dashboard** — Productivity scores, task tracking, focus hours
- 📅 **Smart Planner** — AI-optimized timetable generation
- 📈 **Weekly Analytics** — Charts, progress stats, AI productivity insights
- 🌙 **Dark futuristic UI** — Premium SaaS design with animations

## Tech Stack
